# Clustering Algorithm Performance Visualization

This project visualizes and compares the performance of four clustering algorithms:

- DBSCAN  
- SSA-DBSCAN  
- AMD-DBSCAN  
- k-means  

The script generates:
1. A grouped bar plot that compares all algorithms across numerical metrics.
2. Individual plots for each algorithm with qualitative annotations.
